
The implementation is found in the Main Implementation folder, All html files and python files can be found there




Instructions on how to run:

If you haven't installed python on your machine firstly install version 3.8 from the official python website:

https://www.python.org/downloads/

If you already have python installed it is pretty straight forward:

Open a command prompt and type the following;

!pip install pandas
!pip install flask

Open the a command prompt in the Implementation folder and type;

python main.py


your flask server should now be set up. Leave this open and type the following into your browsers search bar:

http://127.0.0.1:5000/patients

You should now be on our E-Health System!
